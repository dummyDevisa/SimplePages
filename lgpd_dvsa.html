Política de Privacidade – Tratamento de Dados Pessoais para Certificação de Manipuladores de Alimentos

Dados pessoais tratados

A Secretaria Municipal de Saúde (SESMA) realiza o tratamento dos seguintes dados pessoais para fins de certificação de manipuladores de alimentos:

  • Nome completo
  • Número do CPF
  • Situação do CPF
  • Registro Geral (RG)
  • Idade
  • Foto
  • Número de telefone

Finalidade do tratamento

O tratamento de dados pessoais visa garantir a certificação dos manipuladores de alimentos, conforme as normas de saúde pública e segurança alimentar.

Previsão legal do tratamento

O tratamento dos dados pessoais é realizado com base na legislação específica de saúde pública e regulamentações municipais e federais. O fundamento legal será inserido posteriormente.

Compartilhamento de dados

Os dados coletados são compartilhados com outras instituições públicas ou privadas, exclusivamente no que se refere à emissão do certificado de manipulador de alimentos. Este certificado é necessário para comprovação de aptidão dos profissionais no exercício de suas atividades.

Transferência internacional de dados

Amparado pelo Artigo V da LGPD, o DEVISA realiza a transferência internacional dos dados criptografados para os servidores de armazenamento da Google Cloud, por meio da empresa Tally BV (link https://tally.so/help/data-processing-agreement), sediada na Bélgica (União Europeia). Ambas empresas são GDPR compliant, atuando somente como storage database. Todos os dados são permanentemente excluídos dos servidores de armazenamento a cada início de ano-exercício, sem a possibilidade de recuperá-los.

Além disso, o DEVISA também utiliza os serviços do Google Workspace para fins de armazenamento e processamento dos dados. Os provedores de serviço foram selecionados com base na sua capacidade de proteger os dados conforme padrões aceitáveis de compliance e segurança.

Prazo de retenção dos dados

Os dados pessoais serão mantidos pela SESMA enquanto a política pública de certificação de manipuladores de alimentos estiver em vigor, respeitando os prazos legais de armazenamento e a necessidade de preservação para auditoria e controle.

Direitos dos titulares dos dados

Os titulares dos dados têm o direito de acessar, corrigir, excluir ou solicitar a portabilidade de seus dados pessoais, bem como limitar ou opor-se ao seu tratamento. Para exercer esses direitos, os interessados devem contatar a SESMA por meio dos canais disponíveis.

Contato

Caso tenha dúvidas ou queira mais informações sobre o tratamento de seus dados pessoais, entre em contato com a SESMA.
